# SNAE-RCA V7 Enterprise

Architecture propre pour repartir sur une V7 professionnelle.

Modules inclus :
- Portail par thèmes
- Démarches en ligne
- Paiement
- Reçu imprimable
- Tableau de bord manager
- Statistiques
- Routes V7
- Migrations V7
- Seeders V7

Installation :
1. Copier les dossiers dans ton projet Laravel.
2. Lancer : php artisan migrate --seed
3. Lancer : php artisan optimize:clear
